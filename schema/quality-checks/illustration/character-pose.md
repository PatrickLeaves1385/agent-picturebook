# 角色姿势朝向检测

专项检查插图描述中每个出场角色的姿势与朝向。

## 量化锚点（v2 新增）

| 信号 | 关键词 |
|---|---|
| 姿势描写 | 站立/坐着/蹲下/侧身/仰头/低头（至少 1 个） |
| 朝向描写 | 面朝观众/侧身朝左/侧身朝右/仰头朝上/低头朝下（至少 1 个） |
| POV 主观视角 | 第一人称/POV/视角（按需） |

### 单角色判定

| 情况 | 判定 |
|---|---|
| 姿势 + 朝向 都有 | 该角色 PASS |
| 缺姿势或缺朝向 | 该角色 WARNING |
| 缺姿势且缺朝向 | 该角色 FAIL |

### 全场判定

| 情况 | 判定 |
|---|---|
| PASS 角色 / 出场角色 >= 80% | ✅ PASS |
| 50-80% | ⚠️ WARNING |
| < 50% | ❌ FAIL |
| 远景非对话路人角色 | 豁免（不计入分母） |

## 检测方法

1. 从 `wiki/projects/{project_id}/characters.md` 获取已知角色名列表
2. Grep 定位每页插图描述中各角色名
3. 在角色名邻近 50 字符内搜索姿势+朝向关键词
4. 按量化规则判定每角色 + 全场

## 验收标准（原）

### 姿势描写（至少一种）
- 站立 / 坐着 / 蹲下 / 侧身 / 仰头 / 低头 等

### 朝向描写（至少一种）
- 面朝观众 / 侧身朝左 / 侧身朝右 / 仰头朝上 / 低头朝下
- POV 主观视角需标注

### 判定（原）
- 所有出场角色有完整姿势+朝向 → PASS
- 部分角色缺失 → WARNING，标注缺失角色和页码

## 注意
- 远景中的路人角色（非对话参与）可豁免
- 同一角色连续多页出现时，至少有一页完整描写
- 量化后更严：原"部分缺失 WARNING"现按缺失比例分档
