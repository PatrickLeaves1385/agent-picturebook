# 生成参数参考（Step 3.5 引用）

本文档是 `picturebook-art-agent` Step 3.5 的参数数据来源。Agent 在询问参数时引用本文档中的各表，不内联拷贝。

---

## 一、模型选择（二选一）

| 模型 | 特点 | 比例控制 | 分辨率控制 |
|---|---|---|---|
| `gpt-image-2` | OpenAI 兼容，PNG 输出，比例精准 | 输入具体像素（如 1024×768） | 固定（由像素决定） |
| `nano-banana-2` | Gemini 原生，水彩风强，JPEG 输出 | 输入比例（如 `4:3`） | `1K` / `2K` / `4K` 三档 |

> ⚠️ 仅支持以上两种。用户输入其他模型名时拒绝并提示重新选择。

---

## 二、宽高比例（十选一）

| 比例 | 常见用途 |
|---|---|
| `21:9` | 超宽屏 |
| `16:9` | 宽屏 |
| `5:4` | 接近方形（横） |
| `4:3` | 横版绘本 |
| `3:2` | 横版（桌面） |
| `1:1` | 方版绘本 |
| `2:3` | 竖版（手机） |
| `3:4` | 竖版绘本 |
| `4:5` | 接近方形（竖） |
| `9:16` | 竖屏 |

> ⚠️ 仅支持以上十种。用户输入其他比例时拒绝。

**场景建议**（仅供参考，仍须用户确认，禁止默认）：
- `character`：竖版或方版（`3:4` / `1:1`）
- `scene`：横版（`16:9` / `4:3`）
- `prop`：方版（`1:1`）
- `illustration`：按绘本版式

---

## 三、分辨率（三选一）

| 选项 | 说明 |
|---|---|
| `1K`（默认） | 标准清晰度，生成速度快 |
| `2K` | 高清，细节更丰富 |
| `4K` | 超高清，适合印刷 |

---

## 四、画质（仅 gpt-image-2）

| 选项 | 说明 |
|---|---|
| `low` | 快速预览 |
| `medium` | 适中 |
| `high` | 高质量 |
| `auto` | 自动选择 |

nano-banana-2 时跳过此步骤，标注 `[i] nano-banana-2 不支持 quality 参数，跳过画质询问。生成时将传 --quality "auto"（占位，实际无效）。`

---

## 五、gpt-image-2 像素查表

**仅 gpt-image-2 需要**：将用户选择的宽高比例 + 分辨率转为具体像素值。查表来源与 `image-generate` SKILL.md 映射表一致。

| 比例 | 1K 像素 | 2K 像素 | 4K 像素 |
|---|---|---|---|
| `21:9` | 1344×576 | 2016×864 | 3808×1632 |
| `16:9` | 1280×720 | 2048×1152 | 3840×2160 |
| `5:4` | 1040×832 | 2080×1664 | 3200×2560 |
| `4:3` | 1024×768 | 2048×1536 | 3264×2448 |
| `3:2` | 1008×672 | 2064×1376 | 3504×2336 |
| `1:1` | 1024×1024 | 2048×2048 | 2880×2880 |
| `2:3` | 672×1008 | 1376×2064 | 2336×3504 |
| `3:4` | 768×1024 | 1536×2048 | 2448×3264 |
| `4:5` | 832×1040 | 1664×2080 | 2560×3200 |
| `9:16` | 720×1280 | 1152×2048 | 2160×3840 |

**查表示例**：`4:3` + `2K` → `2048×1536`。
